Comparison of note taking tools

Vnote

  • +has vim mode
  • +rendering is different
  • +can add templates, css and code block style
  • +markdown support
  • +navigation bar
  • +can tag files
  • +tabbar
  • Will not automatically detect files in its folder. Have to be present in vnote.json
  • -No way to go to edit mode on the current line

Trillium

  • -Sqlite database, not plain text
  • -Electron
  • +can tag files
  • +Programmable
  • +Link maps

Qownnotes

  • Installation was stuck in AUR
  • +Separate Render window
  • +can tag files
  • -Involves sqlite for tagging
  • -Failed to import existing notes

VS Code

  • +Can open any folder from Terminal
  • +Already used for other purposes
  • -Cannot tag notes
  • -Electron
  • -No templating

Org mode

  • -Cannot be used in mobile
    • Emacsclient is not working anymore in termux.
  • +Best to manage Areas with orgzly

Emacs for markdown

  • rg integration
  • Extendable
  • Keyboard driven

Tiddlywiki

  • NodeJS cannot be used in termux due to huge startup times.
