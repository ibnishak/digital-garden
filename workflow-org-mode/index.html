Workflow: Org mode and Orgzly

Org mode is used for the following things

  • As a linkbox/pocket alternative. I have separate files for videos to watch, articles to read, golang related links.
  • As a shopping list.
  • As a to-do list.
  • As a project tracker. Currently active project gets a separate file. All others are kept in project.org.
  • As an inbox.
